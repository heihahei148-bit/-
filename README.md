AI 冰鉴神相 (AI-Bingjian-Oracle) 项目文档
1. 项目概述
“AI 冰鉴神相”是一个融合了传统东方相术精髓与现代多模态人工智能的 Micro-SaaS 产品。该项目基于曾国藩《冰鉴》中的识人法则，通过分析用户的面部特征、手掌纹路及生辰八字，为用户提供科学化、结构化的命理参考与性格分析。

2. 核心功能
智能八字排盘： 用户输入公历出生时间，系统自动计算生成四柱（年、月、日、时）八字。

多模态视觉分析： 通过上传面部与手掌照片，利用视觉 AI 技术提取特征，并严格映射《冰鉴》相学理论。

中式新美学交互： 采用移动端优先的极简设计，提供沉浸式的测算体验。

自动化交付： 测算结果通过结构化输出，直接展示在前端界面。

3. 技术栈
前端： Vue 3 (Composition API)、Tailwind CSS (响应式移动端适配)、Pinia (状态管理)。

后端： Node.js (Express)、Multer (图片处理)。

数据库： SQLite (用于存储用户基础测算数据)。

关键工具： lunar-javascript (八字计算)。

AI 引擎： 支持接入 GPT-4o 或 Gemini 1.5 Pro 等多模态视觉大模型。

4. 业务逻辑流程
数据采集： 前端采集用户的姓名、生辰、面部及手相照片。

特征处理：

后端将时间数据传给 lunar-javascript 进行排盘。

照片经过预处理后，构建基于《冰鉴》标准的 System Prompt。

大模型推理： 将特征信息与 prompt 封装发送至视觉大模型，模型执行“望、闻、问、切”式的视觉分析。

结果输出： 系统将解析后的报告存入 SQLite 数据库，并同步在移动端展示。

7. 后续开发建议 (Roadmap)

 阶段一： 接入真实的大模型 API，优化 Prompt 以获得更精准的《冰鉴》风格输出。

阶段二： 开发支付集成（如 Stripe 或微信支付），实现“解锁完整报告”的商业闭环。

阶段三： 增加社交裂变功能，允许用户生成精美的“好运签”图片进行朋友圈分享，实现病毒式传播。
